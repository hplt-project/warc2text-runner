##  ႏို၀င္ဘာ ၁၀ ရက္ ည ၉ နာရီ
တုိင္းေဒသႀကီးလႊတ္ေတာ္ မဲရလဒ္
၂၀၁၅ အေထြေထြေရြးေကာက္ပြဲမွာ
အႏိုင္ရခဲ့ၾကတဲ့
တုိင္းေဒသႀကီး/ျပည္နယ္လႊတ္ေတာ္
၄၈ ေနရာအတြက္
ကိုယ္စားလွယ္ေလာင္းေတြရဲ႕

ျပည္ေထာင္စု

(အန္အယ္လ္ဒီ) ၃။ ကခ်င္ျပည္နယ္ ဖားကန္႔
၁ ဦးHလေအာင္ (အန္အယ္လ္ဒီ) ၄။
